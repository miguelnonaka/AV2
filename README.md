## Como executar o projeto

### Passos

1. Clone o repositório:

2. Acesse a pasta do projeto:
   ```bash
   cd aeronave
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Execute o projeto em modo de desenvolvimento:
   
   ```bash
   cd src
   ```
   ```bash
   cd app
   ```
   ```bash
   npm run dev
   ```

6. Abra no navegador:
   ```
   http://localhost:3000
   ```
